# GitHub 账号创建超详细指南（0基础）

## 📝 准备工作

创建 GitHub 账号需要：
- ✅ 一个有效的邮箱地址
- ✅ 一个密码（建议包含大小写字母、数字、符号）
- ✅ 一个用户名（英文字母和数字）

**提示：你可以使用你现有的邮箱：zxqyhm@163.com**

---

## 🚀 创建步骤（5分钟）

### 步骤 1：访问 GitHub 官网

1. **打开浏览器**
   - Chrome、Edge、Firefox 都可以

2. **访问 GitHub 官网**
   - 输入网址：https://github.com
   - 或者直接搜索"GitHub"

---

### 步骤 2：点击注册

1. **找到注册入口**
   - 页面右上角有一个绿色的"Sign up"按钮
   - 或者点击"Sign up"链接

---

### 步骤 3：填写注册信息

**会看到以下表单：**

#### 1. 输入邮箱
- Email address: `zxqyhm@163.com`（或者你其他邮箱）
- 点击"Continue"

#### 2. 设置密码
- Create a password: 输入你的密码
  - 建议：至少8个字符
  - 包含大小写字母、数字
  - 例如：`Lingo123!@#`
- 点击"Continue"

#### 3. 选择用户名
- Enter a username: 输入你想用的用户名
  - 只能使用英文字母、数字、下划线
  - 例如：`lingo2024`、`languagelearner`
  - 如果用户名被占用，GitHub会提示你换一个
- 点击"Continue"

#### 4. 确认邮箱订阅（可选）
- 是否接收产品更新
- 你可以选择取消勾选（不想接收）
- 点击"Continue"

---

### 步骤 4：验证人机（可选）

**可能会出现一个拼图验证：**
- 拖动滑块完成拼图
- 或者选择包含某种物体的图片
- 点击"Verify"

---

### 步骤 5：验证邮箱（重要！）

1. **检查邮箱**
   - 打开你的邮箱：zxqyhm@163.com
   - 找到 GitHub 发送的验证邮件
   - 标题通常是："Please verify your email address"

2. **打开邮件**
   - 点击邮件内容中的"Verify email address"按钮
   - 或者复制验证链接到浏览器

3. **验证成功**
   - 会显示"Email verified"
   - 自动跳转到 GitHub 主页

**注意：如果没收到邮件：**
- 检查垃圾邮件文件夹
- 等待 1-2 分钟
- 或者在 GitHub 点击"Resend verification email"（重新发送）

---

### 步骤 6：完成注册

验证邮箱后：
1. 会自动登录到 GitHub
2. 看到欢迎页面
3. 创建成功！

## ❓ 遇到问题？

### 常见问题：

1. **"用户名已被占用"**
   - 换一个用户名
   - 加一些数字，如 `lingo2024`、`lingo2025`

2. **"邮箱已被注册"**
   - 说明你之前注册过
   - 点击"Sign in"直接登录
   - 或者使用"Forgot password"重置密码

3. **"验证邮件没收到"**
   - 检查垃圾邮件文件夹
   - 点击"Resend verification email"重新发送
   - 确保邮箱地址正确

4. **"拼图验证失败"**
   - 刷新页面重试
   - 或者使用其他验证方式
