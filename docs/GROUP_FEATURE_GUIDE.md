# 群组功能使用指南

## 🎉 功能概述

Lingo 应用现已支持完整的微信群组功能，包括：

### ✅ 核心功能

1. **创建群组**
   - 设置群名称、群描述
   - 自定义群组设置（截屏、录屏、公开/私密）
   - 自动添加群主

2. **群组聊天**
   - 实时消息发送和接收
   - 消息按时间排序
   - 区分自己和他人的消息

3. **群组管理**
   - 查看群组成员列表
   - 退出群组
   - 群主专属设置

4. **安全功能** 🛡️
   - **禁止截屏**：群主可设置禁止成员截屏群聊内容
   - **禁止录屏**：群主可设置禁止成员录屏群聊内容
   - **私密群组**：设置后只有群主邀请才能加入

---

## 📱 如何使用

### 1. 创建群组

1. 进入 **消息** 页面
2. 点击右上角 `+` 按钮
3. 填写群组信息：
   - **群名称**（必填）
   - **群描述**（可选）
4. 设置群组安全选项：
   - **允许截屏**：关闭后成员无法截屏
   - **允许录屏**：关闭后成员无法录屏
   - **公开群**：关闭后只有邀请才能加入
5. 点击 **创建** 按钮

### 2. 加入群组

- 点击群组列表中的任意群组
- 如果是公开群，可以直接加入
- 如果是私密群，需要群主邀请（功能开发中）

### 3. 群组聊天

1. 进入群组后，底部有输入框
2. 输入消息内容，点击发送图标
3. 消息会实时显示在聊天列表中

### 4. 群主设置

1. 进入群组聊天页面
2. 点击右上角 **设置** 图标
3. 修改群组安全设置：
   - 切换 **允许截屏** 开关
   - 切换 **允许录屏** 开关
4. 点击 **保存设置**

## 🛡️ 安全功能说明

### 禁止截屏/录屏的工作原理

1. **Android 设备**：
   - 使用 `FLAG_SECURE` 标志
   - 截屏时会显示黑色屏幕
   - 录屏时会显示黑屏或无内容

2. **iOS 设备**：
   - 系统限制较多
   - 部分设备可能无法完全阻止截屏
   - 建议配合其他安全措施使用

### 安全提示

- ⚠️ 截屏/录屏保护不是 100% 可靠
- ⚠️ 用户仍可通过其他方式（如拍照）获取内容
- ⚠️ 建议在讨论敏感话题时提醒成员保护隐私
- ✅ 最佳实践：定期更改敏感信息，使用端到端加密（后续版本）

## 🔧 API 接口

### 群组相关接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/groups` | POST | 创建群组 |
| `/api/v1/groups` | GET | 获取群组列表 |
| `/api/v1/groups/:id` | GET | 获取群组详情 |
| `/api/v1/groups/:id/join` | POST | 加入群组 |
| `/api/v1/groups/:id/leave` | POST | 退出群组 |
| `/api/v1/groups/:id/settings` | POST | 更新群组设置 |
| `/api/v1/groups/:id/messages` | GET | 获取群消息 |
| `/api/v1/groups/:id/messages` | POST | 发送群消息 |

## 📋 数据库表结构

### groups - 群组表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | varchar(36) | 群组 ID |
| name | varchar(100) | 群名称 |
| avatar_url | varchar(500) | 群头像 |
| description | text | 群描述 |
| owner_id | varchar(36) | 群主 ID |
| member_count | integer | 成员数量 |
| max_members | integer | 最大成员数（默认 500）|
| allow_screenshot | boolean | 是否允许截屏 |
| allow_screen_recording | boolean | 是否允许录屏 |
| is_public | boolean | 是否公开群 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

### group_members - 群成员表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | varchar(36) | 成员 ID |
| group_id | varchar(36) | 群组 ID |
| user_id | varchar(36) | 用户 ID |
| role | varchar(20) | 角色（owner/admin/member）|
| nickname | varchar(64) | 群昵称 |
| muted_until | timestamp | 禁言到期时间 |
| joined_at | timestamp | 加入时间 |

### group_messages - 群消息表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | varchar(36) | 消息 ID |
| group_id | varchar(36) | 群组 ID |
| sender_id | varchar(36) | 发送者 ID |
| content | text | 消息内容 |
| type | varchar(20) | 消息类型（text/image/voice/video）|
| media_url | varchar(500) | 媒体 URL |
| reply_to_id | varchar(36) | 回复的消息 ID |
| is_read | jsonb | 已读状态 |
| created_at | timestamp | 发送时间 |
